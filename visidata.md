# visidata

## 移動
| キー | 操作 |
|---|---|
| `h/j/k/l` | 左/下/上/右 |
| `gj` / `gk` | 末尾行/先頭行 |
| `gh` / `gl` | 先頭列/末尾列 |
| `PageDown/Up` | ページ単位で移動 |
| `zj` / `zk` | 画面下端/上端 |

## 検索・フィルタ
| キー | 操作 |
|---|---|
| `/` | 正規表現で前方検索 |
| `?` | 正規表現で後方検索 |
| `n` / `N` | 次/前の検索結果 |
| `\|` | 選択行をフィルタ（新シート） |

## ソート
| キー | 操作 |
|---|---|
| `[` | 昇順ソート |
| `]` | 降順ソート |

## 列操作
| キー | 操作 |
|---|---|
| `-` | 列を非表示 |
| `_` | 列幅を内容に合わせる |
| `g_` | 全列の幅を調整 |
| `!` | 列をキー列に設定 |
| `#` | 列を整数型に |
| `%` | 列を浮動小数型に |
| `~` | 列を文字列型に |
| `^` | 列名を変更 |
| `=` | 新しい列を式で追加 |
| `H` / `L` | 列を左/右に移動 |

## 行操作
| キー | 操作 |
|---|---|
| `a` | 行を追加 |
| `e` | セルを編集 |
| `d` | 行を削除 |
| `gd` | 選択行を一括削除 |
| `J` / `K` | 行を下/上に移動 |

## 行選択
| キー | 操作 |
|---|---|
| `s` / `u` | 行を選択/解除 |
| `gs` / `gu` | 全行選択/解除 |
| `,` | 選択行だけの新シート |
| `"` | 選択行のコピーシート |

## シート操作
| キー | 操作 |
|---|---|
| `S` | シート一覧 |
| `C` | 列一覧 |
| `F` | 現在の列で頻度表 |
| `I` | 列の統計情報（平均・合計等） |
| `T` | 転置（行と列を入れ替え） |
| `.` | 現在のセルを展開 |

## コピー・ペースト
| キー | 操作 |
|---|---|
| `y` | 選択行をコピー |
| `gy` | 現在の列の値をコピー |
| `p` | コピーした行を貼り付け |
| `zd` | セルの値をクリア |

## 集計（キー列設定後）
| キー | 操作 |
|---|---|
| `+` | 集計関数を追加（sum, avg 等） |
| `z+` | 選択行に集計関数を適用 |

## 保存・終了
| キー | 操作 |
|---|---|
| `Ctrl+s` | 保存（形式はファイル拡張子で決定） |
| `q` | 現在のシートを閉じる |
| `gq` | 全終了 |

## ヘルプ
| キー | 操作 |
|---|---|
| `z Ctrl+h` | 現在のキーバインド一覧 |
