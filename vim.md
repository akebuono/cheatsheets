# vim

## 移動
| キー | 操作 |
|---|---|
| `h/j/k/l` | 左/下/上/右 |
| `w` / `b` | 単語単位で前/後 |
| `e` | 単語末尾 |
| `0` / `$` | 行頭/行末 |
| `^` | 行頭（空白除く） |
| `gg` / `G` | ファイル先頭/末尾 |
| `10G` | 10行目にジャンプ |
| `Ctrl+d/u` | 半ページ下/上 |
| `Ctrl+f/b` | 1ページ下/上 |
| `%` | 対応する括弧にジャンプ |
| `f{char}` | 行内で文字を前方検索 |
| `t{char}` | 行内で文字の手前に移動 |

## 編集
| キー | 操作 |
|---|---|
| `i` / `a` | カーソル前/後に挿入 |
| `I` / `A` | 行頭/行末に挿入 |
| `o` / `O` | 下/上に行追加 |
| `x` | 1文字削除 |
| `r{char}` | 1文字置換 |
| `R` | 上書きモード |
| `c{motion}` | 削除して挿入（`cw`=単語、`c$`=行末まで） |
| `C` | カーソルから行末まで削除して挿入 |
| `s` | 1文字削除して挿入 |
| `S` / `cc` | 行全体を削除して挿入 |
| `J` | 下の行を結合 |
| `.` | 直前の操作を繰り返し |

## カット・コピー・ペースト
| キー | 操作 |
|---|---|
| `dd` | 行カット |
| `3dd` | 3行カット |
| `dw` | 単語カット |
| `d$` / `D` | 行末までカット |
| `d0` | 行頭までカット |
| `yy` | 行コピー |
| `3yy` | 3行コピー |
| `yw` | 単語コピー |
| `p` / `P` | カーソル後/前に貼り付け |
| `"ayy` | レジスタ a にコピー |
| `"ap` | レジスタ a から貼り付け |

## ビジュアルモード（選択）
| キー | 操作 |
|---|---|
| `v` | 文字単位で選択開始 |
| `V` | 行単位で選択開始 |
| `Ctrl+v` | 矩形選択（ブロック選択） |
| `gv` | 前回の選択範囲を再選択 |

### 矩形選択の操作
| 手順 | 操作 |
|---|---|
| `Ctrl+v` → 範囲選択 → `d` | 矩形削除 |
| `Ctrl+v` → 範囲選択 → `y` | 矩形コピー |
| `Ctrl+v` → 範囲選択 → `I` → 入力 → `Esc` | 矩形の先頭に一括挿入 |
| `Ctrl+v` → 範囲選択 → `A` → 入力 → `Esc` | 矩形の末尾に一括挿入 |
| `Ctrl+v` → 範囲選択 → `r{char}` | 矩形を1文字で一括置換 |
| `Ctrl+v` → 範囲選択 → `c` → 入力 → `Esc` | 矩形を一括変更 |

## 検索・置換
| キー | 操作 |
|---|---|
| `/word` | 前方検索 |
| `?word` | 後方検索 |
| `n` / `N` | 次/前 |
| `*` / `#` | カーソル下の単語を前方/後方検索 |
| `:s/old/new/` | 現在行の最初を置換 |
| `:s/old/new/g` | 現在行の全てを置換 |
| `:%s/old/new/g` | ファイル全体を置換 |
| `:%s/old/new/gc` | 確認しながら全置換 |

## インデント
| キー | 操作 |
|---|---|
| `>>` / `<<` | 行のインデント追加/削除 |
| `V` → 選択 → `>` / `<` | 選択範囲のインデント |
| `=` | 自動インデント |
| `gg=G` | ファイル全体を自動インデント |

## ウィンドウ分割
| キー | 操作 |
|---|---|
| `:sp` | 水平分割 |
| `:vsp` | 垂直分割 |
| `Ctrl+w h/j/k/l` | ウィンドウ移動 |
| `Ctrl+w q` | ウィンドウを閉じる |

## マクロ
| キー | 操作 |
|---|---|
| `qa` | レジスタ a に記録開始 |
| `q` | 記録終了 |
| `@a` | マクロ a を実行 |
| `10@a` | マクロ a を10回実行 |

## 保存・終了
| キー | 操作 |
|---|---|
| `:w` | 保存 |
| `:q` | 終了 |
| `:wq` / `ZZ` | 保存して終了 |
| `:q!` / `ZQ` | 保存せず終了 |
| `:wa` | 全バッファ保存 |
| `:qa` | 全終了 |
