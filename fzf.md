# fzf

## 基本
| コマンド | 説明 |
|---|---|
| `fzf` | カレント以下のファイルを検索 |
| `cat file \| fzf` | パイプで絞り込み |
| `vim $(fzf)` | 選択したファイルを vim で開く |
| `bat $(fzf)` | 選択したファイルを bat で表示 |

## TUI 操作
| キー | 操作 |
|---|---|
| 入力 | インクリメンタル検索 |
| `↑/↓` | 候補移動 |
| `Enter` | 選択 |
| `Tab` | 複数選択 |
| `Ctrl+c` | キャンセル |

## 検索構文
| パターン | 意味 |
|---|---|
| `word` | あいまい検索 |
| `'word` | 完全一致 |
| `^word` | 前方一致 |
| `word$` | 後方一致 |
| `!word` | 除外 |
